## 使用

在VeighNa中使用Tdengine时，需要在全局配置中填写以下字段信息：

|名称|含义|必填|举例|
|---------|----|---|---|
|database.name|名称|是|tdengine|
|database.host|地址|是|localhost|
|database.port|端口|是|6041|
|database.database|实例|是|vnpy|
|database.user|用户名|是|root|
|database.password|密码|是|taosdata|
